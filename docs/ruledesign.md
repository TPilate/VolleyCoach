## 🍏 Apple Liquid Glass — Designer Reference

**Announced:** WWDC 2025  
**Platforms:** iOS 26, macOS Tahoe 26, etc.

### What is it?
A **dynamic material system** combining:
- Translucency
- Real-time refraction
- Motion responsiveness

---

### Key Features

#### ✨ Translucency & Refraction
- Background bends through layers  
- GPU-accelerated rendering  

#### ♿ Accessibility Built-In
- Minimum contrast ratio: **4.5:1**  
- Supports reduced transparency  

#### 🌍 Universal System
- Unified design across all Apple platforms  

---

### Recommended Settings (Figma)

| Property | Value | Notes |
|--------|------|------|
| Depth | 0–20 | >20 harms readability |
| Frost | 10–25 | >30 looks artificial |
| Contrast | ≥ 4.5:1 | Required |
| Layers | 1 | Avoid stacking |

---

### ✅ Do / ❌ Don’t

**Do**
- Use for navigation elements  
- Keep it functional  
- Ensure accessibility  

**Don’t**
- Stack multiple glass layers  
- Use on scrollable content  
- Overuse tinting

## 🧠 Core UX/UI Principles

### Visual Hierarchy
- Use size, contrast, spacing  
- Design for **F/Z scanning patterns**

---

### Cognitive Load
- Hick’s Law: fewer choices = faster decisions  
- Use progressive disclosure  

---

### Consistency
- Follow familiar patterns (Jakob’s Law)  
- Keep UI elements uniform  

---

### Accessibility (WCAG)
- Target **WCAG AA** minimum  
- 4.5:1 contrast ratio  
- Keyboard + screen reader support  

---

### Performance
- 1-second delay reduces conversions  
- Mobile-first (62.5% of traffic)  

---

### Feedback & States
- Always show system response  
- Use loading, success, error states  

---

### Responsive Design
- Start mobile-first  
- Minimum touch target: **44×44px**  

---

### Test & Iterate
- Fixing UX late = 10× cost  
- Use usability testing + heatmaps  

---

### Design Systems
- Ensure consistency  
- Document components & usage

## 🎨 Creative Trends 2025–2026

### Bento Grid Layouts
- Asymmetrical card layouts  
- Structured but dynamic  

---

### Neo-Brutalism
- Raw, bold, unconventional  
- Clashing colors, big typography  

---

### Kinetic Typography
- Motion-driven text  
- Replaces static visuals  

---

### Glassmorphism / Liquid Glass
- Frosted layers + vibrant backgrounds  
- Must maintain accessibility  

---

### Organic Layouts
- Fluid shapes, broken grids  
- More human-centered design  

---

### Gamification & Scroll Storytelling
- Interactive journeys  
- Progress-based navigation  

---

### Custom Cursors
- Branded interactive cursors  
- Becoming standard in portfolios  

---

### Nature-Inspired Palettes
- Earth tones (clay, wood, soil)  
- Calm, sustainable aesthetics
